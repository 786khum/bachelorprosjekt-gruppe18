# bachelorprosjekt-gruppe18

Dette prosjektet utvikler en KI basert chatbot for Testfest.no i samarbeid med Tingtun. Chatboten skal hjelpe tjenesteeiere med universell utforming ved å svare på spørsmål, foreslå testoppgaver og presentere funn på en forståelig måte.
Prosjektet bruker Gemini som KI modell.
Teknologi brukt ellers: GitHub, Figma og Scrum basert arbeidsflyt.
Status: Under utvikling. MVP leveres våren 2026.
